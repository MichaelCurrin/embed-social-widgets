Instagram embed

Use Instagram's native embed functionality for single posts.

Approach

Use URl for IG post with /embed appended and add it as the "src" for an iframe.

I recommend setting width and height as well to avoid getting a box that is cut off unnaturally.

Code sample: 

    <iframe
        src="https://www.instagram.com/p/POST_ID/embed"
        width="XXX" height ="YYY">
    </iframe>
